Android Apps

Here are some of the Android apps I have developed.

Android's YouTube App

A YouTube clone app for Android built with Jetpack Compose. It allows users to browse and watch videos. Features include search, play in the background, playlists, and more.
Languages: Kotlin for frontend and Python for backend.
Frameworks: Jetpack Compose for frontend.
Tools: Android Studio for development.

Music Player App

A music player app for Android built with Jetpack Compose. It allows users to play and manage their music library. Features include playlists, equalizer settings, and background playback.
Languages: Kotlin.
Frameworks: Jetpack Compose for frontend.
Tools: Android Studio for development.

QR Code Scanner App & Generator

A QR code scanner and generator app for Android built with Jetpack Compose. It allows users to scan and generate QR codes. Features include history of scanned codes and sharing options.
Languages: Kotlin.
Frameworks: Jetpack Compose for frontend.
Tools: Android Studio for development.
